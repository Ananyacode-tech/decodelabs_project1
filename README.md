# To-Do List(Python)

## Description 
A simple command-line To-Do List application built in Python. 
This project allows users to manage their daily tasks while practicing Python fundamentals such as loops, conditional statements and lists.

## Features 
-Add new tasks
-View all tasks
-Exit task
-Menu-driven interface using loops and if-else statements

## Technologies Used
-Python

## How to Run
1. Make sure Python is installed.
2. Download or clone this repository.
3. Run the preoject.

## Concepts Used
-Loops("While")
-Conditional statements('if-else')
-Lists
-User input('input()')
